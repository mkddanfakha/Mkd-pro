# Vidéos des Projets

Placez ici les vidéos de démonstration de vos projets.

## Vidéos pour l'application de gestion de stock et vente

Deux vidéos sont nécessaires pour ce projet :

1. **Vidéo en français** : `stock-vente-demo-fr.mp4`
2. **Vidéo en wolof** : `stock-vente-demo-wo.mp4`

Placez vos deux vidéos dans ce dossier avec les noms exacts ci-dessus.

### Formats acceptés
- MP4 (recommandé - meilleure compatibilité)
- WebM
- OGG
- MOV

### Recommandations
- Résolution : 1920x1080 (Full HD) ou 1280x720 (HD)
- Codec vidéo : H.264 (pour MP4)
- Codec audio : AAC (pour MP4)
- Durée : 30-60 secondes recommandé pour une démo
- Poids : < 10MB (optimisez la vidéo si nécessaire)

### Optimisation
Pour réduire la taille de la vidéo, vous pouvez utiliser des outils comme :
- HandBrake (gratuit)
- FFmpeg (ligne de commande)
- Online tools : CloudConvert, FreeConvert
