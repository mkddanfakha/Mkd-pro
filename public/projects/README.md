# Images des Projets

Placez ici les images mockup de vos projets.

## Image pour l'application de gestion de stock et vente

Nom du fichier : `stock-vente-mockup.png`

Placez votre image mockup dans ce dossier avec le nom exact : `stock-vente-mockup.png`

### Formats acceptés
- PNG (recommandé)
- JPG/JPEG
- WebP

### Taille recommandée
- Largeur : 1200px minimum
- Ratio : 16:9 ou 4:3
- Poids : < 500KB (optimisez l'image si nécessaire)
